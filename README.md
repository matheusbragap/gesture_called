# Gesture Called

O Gesture Called reúne a aplicação ServFlow, criada para centralizar a abertura, o acompanhamento e a resolução de chamados técnicos e operacionais entre lojas e equipe de suporte.

## Visão do Projeto

Em empresas com múltiplas unidades, a comunicação de suporte costuma ficar dispersa em vários canais. O Gesture Called organiza esse fluxo em um único sistema, mantendo cada chamado vinculado à unidade de origem para garantir rastreabilidade e histórico.

## Stack

- Dart
