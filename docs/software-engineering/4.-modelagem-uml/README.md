* [4.3 Sequência](4.3-sequencia.md)

### Organização adotada

* Os casos de uso apresentam as interações entre os atores e as funcionalidades do sistema.
* O diagrama de classes apresenta a estrutura das entidades principais e seus relacionamentos.
* Os diagramas de sequência apresentam a ordem das interações em fluxos relevantes.
* O diagrama de estados apresenta as mudanças de status do chamado ao longo do atendimento.
