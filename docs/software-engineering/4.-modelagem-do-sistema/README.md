# 4. Modelagem do Sistema

### Estrutura da seção

* [4.1 Casos de Uso](4.1-diagrama-de-casos-de-uso-uml.md)
* [4.2 Diagrama de Classes](4.2-diagrama-de-classes-uml.md)

### Organização adotada

* Os casos de uso apresentam as interações entre os atores e as funcionalidades do sistema.
* O diagrama de classes apresenta a estrutura das entidades principais e seus relacionamentos.
* O diagrama entidade-relacionamento apresenta a estrutura lógica dos dados e suas associações.
* O diagrama de fluxo de dados apresenta a circulação de informações entre usuários, processos e armazenamento.
