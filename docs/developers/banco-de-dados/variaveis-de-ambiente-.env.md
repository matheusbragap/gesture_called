# Variáveis de Ambiente (.env)

Centralize nesta página todas as variáveis de ambiente do projeto.

### Para cada variável, informe

* nome
* descrição
* obrigatória ou opcional
* valor de exemplo
* ambiente de uso
* impacto se estiver ausente

### Sugestão de grupos

* banco de dados
* autenticação
* serviços externos
* URLs da aplicação
* chaves e segredos
* flags de ambiente
