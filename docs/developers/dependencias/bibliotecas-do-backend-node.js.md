# Bibliotecas do Backend (Node.js)

Use esta página para centralizar as dependências do backend.

### Base

O backend usa **Node.js** e **npm**.

* Instale Node.js em [nodejs.org/pt-br/download](https://nodejs.org/pt-br/download)
* Prefira a versão **LTS**
* Valide com:

```bash
node --version
npm --version
```

### Instalação

Na pasta do backend, rode:

```bash
npm install
```

Se quiser uma instalação limpa em ambiente controlado:

```bash
npm ci
```

### Arquivos de referência

Use estes arquivos como fonte de verdade:

* `package.json`
* `package-lock.json`
