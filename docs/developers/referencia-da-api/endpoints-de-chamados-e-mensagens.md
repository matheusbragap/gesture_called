# Endpoints de Chamados e Mensagens

Documente nesta página os endpoints ligados ao domínio principal do sistema.

### Inclua em cada rota

* método HTTP
* caminho
* autenticação exigida
* parâmetros e filtros
* body da requisição
* exemplo de resposta
* regras de negócio
* códigos de erro

### Sugestão de organização

* criação de chamado
* listagem e detalhamento
* atualização de status
* envio de mensagens
* histórico e paginação
